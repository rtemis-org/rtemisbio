# rtemis.bio

Bioinformatics extensions for [rtemis](https://www.github.com/rtemis-org/rtemis)

## Installation

You can install `rtemis.bio` from `r-universe` or using `pak`, `remotes`, or `devtools`.

* `r-universe`:

  ```r
  install.packages('rtemis.bio', repos = c('https://rtemis-org.r-universe.dev', 'https://cloud.r-project.org'))
  ```

* `pak`:

  ```r
  pak::pak("rtemis-org/rtemis.bio")
  ```
